# TrainBuddy

TrainBuddy visualisiert die Daten einer Zugfahrt in einem ÖBB Railjet in einer App, wie z.B. die aktuelle GPS Position des Zuges, nächste Halte, Umstieg Möglichkeiten, aktuelle Geschwindigkeit des Zuges, ...
